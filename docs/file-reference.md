# File Reference

Every file in the QuantForg foundation, explained.

## Repository root

| File | Purpose |
|---|---|
| `README.md` | Project overview, quick start, architecture summary |
| `LICENSE` | Proprietary license placeholder |
| `CHANGELOG.md` | Keep a Changelog release notes |
| `CONTRIBUTING.md` | Contributor setup, workflow, acceptance rules |
| `CODEOWNERS` | Required reviewers by path |
| `pyproject.toml` | Poetry deps, Black/Ruff/MyPy/Pytest/Coverage config |
| `poetry.lock` | Locked dependency versions (generated by `poetry lock`) |
| `.env.example` | Documented environment variable template |
| `.editorconfig` | Cross-editor indentation and charset rules |
| `.pre-commit-config.yaml` | Pre-commit hooks (Ruff, Black, MyPy, secret detection) |
| `.gitignore` | VCS ignore rules |
| `Makefile` | Developer command shortcuts |
| `Dockerfile` | Multi-stage production image build |
| `docker-compose.yml` | Root include that delegates to `docker/docker-compose.yml` |
| `alembic.ini` | Alembic configuration (URL overridden in `env.py`) |
